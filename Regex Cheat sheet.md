#REGEX
```
. - matches any one chacater
+ - matches 1 or more occurances of the last symbol
* - matches 0 or more occurances of the last symbol
{n}, {n,m} - repeat previous match n times or n to m times
() - group characters together
(adc) = adcadcadc

| - matches either pattern on left or right
? - match 0 or 1 occurance
^ - beginning of line
$ - End of line
\b - Word boundary
 Word boundary = \bman will match man " The man" and not " The human".
 
 [abc] matches a, b or c 
 [^12345] MATCH all the characters that are not 12345
 
```
